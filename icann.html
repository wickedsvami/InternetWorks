Domain Names & ICANN

ICANN(Internet Corporation for Assigned Names and Numbers) is an American nonprofit organization that coordinates procedures and regular maintenance of databases that are related to namespaces and numerical spaces of the Internet. Much of the work done by ICANN has to do with the Internet's global Domain Name System(DNS).

While computers use IP addresses for identifying other computers, humans developed an easier way so they could identify other entities on the net, which is a domain name. However, a domain name is not the same thing as a Uniform Resource Locator(URL). A good example to demonstrate this is to think of a domain name as a street address for getting postal mail. There must still be a building or a post office box to be able to receive any mail.
